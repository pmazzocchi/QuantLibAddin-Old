QuantLibXL / ObjectHandler 1.13 Released

QuantLibXL, QuantLibAddin, ObjectHandler, and gensrc version
1.13 have been released and are available for download:
https://bintray.com/quantlib/releases/QuantLibXL/1.13#files

QuantLibAddin
http://www.quantlibaddin.org

QuantLibAddin exports the QuantLib interface to a variety
of end user platforms.

QuantLibXL
http://www.quantlibxl.org

QuantLibXL is the implementation of QuantLibAddin for
Microsoft Excel.  The QuantLibXL project includes a binary
release comprising a compiled Addin and example
workbooks.

ObjectHandler
http://www.objecthandler.org

ObjectHandler implements a repository where objects can be
stored, shared, updated, interrogated, and destroyed.  This
facilitates object orientation in procedural environments
such as spreadsheets.
